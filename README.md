# 开拓轶事 KaiTuoYiShi

开拓轶事是一个基于 **React 19 + TypeScript + Vite + Tailwind CSS** 的同人叙事 Web 应用，目标是把“AI 跑团式剧情生成、世界书、变量系统、伙伴档案、手机、新闻、记忆、智库与文生图资产管理”组织成一个可游玩的前端体验。

当前项目仍处于早期开发阶段，第一阶段重点围绕“登上星穹列车”相关流程打磨。

## 功能概览

- 新游戏向导：创建旅人档案、选择命途、阵营、故事模式与开局设定。
- 主剧情生成：支持流式 / 非流式 AI 回复、回合记录与行动选项。
- 世界书与提示词模块：内置开局规范、变量系统、伙伴档案等规则模块。
- 变量系统：从正文中结算角色、世界、背包、伙伴、手机等结构化状态。
- 伙伴系统：维护 NPC / 伙伴档案、关系、好感、同行记忆、头像与视觉档案。
- 背包与装备：方格背包、装备槽位、物品详情与 AI 写入兜底。
- 记忆系统：即时、短期、长期记忆与 NPC 同行记忆压缩。
- 手机、新闻、剧情编织、智库、相册 / 文生图工作台等扩展系统。

## 本地运行

```bash
npm install
npm run dev
```

默认由 Vite 启动本地开发服务。构建检查：

```bash
npm run build
```

## 项目结构

```text
components/  前端组件与游戏系统界面
data/        内置菜单、提示词、世界书、预设数据
hooks/       游戏状态与主流程工作流
models/      TypeScript 数据模型
prompts/     CoT 与系统提示词模块
public/      随包发布的公开预设资源
services/    AI、存档、剧情、记忆等服务层
styles/      主题与全局样式
utils/       变量执行、背包、世界书、图片等工具函数
```
